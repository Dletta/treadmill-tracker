## Bluetooth Treadmill Tracker

This project is an example on using the Web Bluetooth API to read treadmill data via Bluetooth.

This is for the PitPat-01 Treadmill. Due to the lack of public documentation, currently only reading data is available.

### Usage

Navigate to the Github Pages for this repository, then click on 'Connect to Treadmill' (the treadmill icon button on the bottom left). The browser will then prompt to select the Bluetooth device. 

If your treadmill is compatible it will be listed as 'PitPat-01' or similar.

Once connected, the display will refresh at the rate the treadmill sends the data.

Happy Running!
